Task 1: Cartpole¶

Introduction¶

In this task, you are supposed to balance a pole on a cart for as long as possible. The pole will wiggle back and forth, and the only thing you can do to prevent it from falling, is to move the cart left or right. For example, if the pole is falling towards the right, the cart can counter-balance that by also moving right which will force the pole towards the left.

This is, of course, a delicate equilibrium. Can you keep the pole from falling using spiking neural networks?

Agent inputs and outputs¶

The simulated pendulum provides two measured quantities: current pendulum angle and current angular velocity of the pendulum. Each such quantity is represented by two neurons to avoid negative activations. Therefore, your network receives four inputs: left angle, right angle, left angular velocity, and right angular velocity.

Input

Description

Value range

Left angle

Angular distance towards the left

\([0;0.42]\)

Right angle

Angular distance towards the right

\([0;0.42]\)

Left angular velocity

Angular velocity towards the left

\([0;1]\)

Right angular velocity

Angular velocity towards the right

\([0;1]\)

Output to the environment consists of a single leaky integrator neuron. If the neuron membrane voltage is \(\le\) 0V, the cart moves left. Otherwise, the cart moves right.

\[\begin{split} cart(x) = \begin{cases} \text{left}, & \text{if } x \le 0\\ \text{right}, & \text{otherwise} \end{cases} \end{split}\]

The environment will terminate if the pole falls more than \(0.24\) radians towards the left or right.

How to program your “brain”¶

In the code cell below, you can program your “brain” by defining

  1. a weight matrix that describes synapse connectivity strength and

  2. a neural network that describes how signals flow from top to bottom

Press the ‣ button to execute the code.

from norse_rl.workshop import *

##
## Weight matrix
##
w = [[ 
       1,  # Left angle
       1,  # Right angle
       1,  # Left angular velocity
       1   # Right angular velocity
    ]]

##
## Neural network
##
model = norse.SequentialState(
    norse.LIFCell(),
    Linear(4, 1, w),
    norse.LICell(),
)

##
## Simulate your neural network
##
run("Cartpole-v0", model)
